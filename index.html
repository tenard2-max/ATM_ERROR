📊 전체기기 월별비교 PoC

GitHub Pages에서는 분석 기능이 동작하지 않습니다.
차트·업로드·DB 기능은 Python Flask 서버가 필요합니다. 아래 방법 중 하나로 실행해 주세요.
▶ 배포된 앱 열기 Render 배포 가이드 DEPLOY.md 보기

로컬에서 실행

저장소를 clone한 뒤 프로젝트 폴더에서:

pip install -r requirements.txt
python web_app.py

브라우저에서 http://localhost:8502 접속 (자동으로 열립니다)

배포 URL 연결 (선택)

Render 등에 배포했다면 site-config.js의 liveAppUrl에 URL을 입력하면 GitHub Pages 접속 시 자동으로 이동합니다.
